# Tic-tac-toe Game

A simple browser-based Tic-tac-toe game built with HTML, CSS, and JavaScript.

## Play Now

You can play the game here: [Play Tic-tac-toe](https://victorovi4.github.io/tic-tac-toe/)

## Features

- Clean, modern interface
- Responsive design
- Turn-based gameplay
- Win and draw detection
- Game status display
- Restart functionality
